# 🏋️ Falcon Fitness – Gym Website

A modern, responsive gym website designed to showcase fitness services, membership plans, trainers, and allow potential clients to contact the gym easily.

## 🚀 Live Demo

https://falconfitness.netlify.app/

## 📌 Features

* Modern landing page design
* Hero section with call-to-action
* About section for gym information
* Membership plans section
* Trainers showcase section
* Contact form for customer enquiries
* Smooth scrolling animations
* Fully responsive design (mobile, tablet, desktop)

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Tailwind CSS
* Netlify (for deployment)
